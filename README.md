# 📧 EmailHelper

EmailHelper is an AI-powered email generation application that helps users create professional email replies using Google's Gemini AI. The project includes a Spring Boot backend, a React frontend, and a Chrome Extension for seamless email generation.

## 🚀 Features

- AI-powered email generation
- Multiple email tones
- Chrome Extension integration
- Spring Boot REST API
- Responsive React UI
- Gemini AI integration

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring AI
- React
- Vite
- JavaScript
- Google Gemini API
- Chrome Extension

## 📂 Project Structure

```
backend/
frontend/
chrome-extension/
```

## ⚙️ Run Locally

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
